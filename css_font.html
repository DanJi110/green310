CSS 글자속성

글자에 관련된 속성은 글꼴, 글자크기, 글자굵기, 기울임, 줄 간격, 대소문자, 글자속성 등의 주된 일곱가지를 필수적으로 알고 있어야 한다

color를 제외한 나머지 여섯 가지 속성은 한 줄 쓰기가 가능하다
속성명 속성값 종류 설명
font-family 내컴퓨터에 등록되어 있는 폰트/ 웹폰트 활용가능 글꼴을 인용부호에 감싼다
다국적사용자를 위한 페이지로 사용할시 대체폰트의 의미로, serif(명조, 궁서)/Sans-serif(고딕) 를 작성한다
font-size 단위를 여러개 적용할 수 있으며 16px, 1em 등으로 기본값이 설정되어 있다 숫자로 기입시 필수적으로 단위와 함께 적용하고 기본 크기를 기반으로 대략적인 비율을 형성하여 small, large 등으로 기재가 가능하다
font-style italic / normal 글자의 기울기를 지정 / 기본값이 기울어져 있는 태그의 경우 설정값 변경으로 사용가능하다
font-variant small-caps / normal 작은 대문자로 변경하거나 일반형으로 작성 / 그 외 변형은 추가 속성에서 설명할 예정
font-weight blod/normal 로 쓰거나 숫자 400-nomal, 600 bold 글자의 두께를 지정한다, 서체에 따라 굵기는 다를 수 있다
line-height 기본값 1.4(140%) /
단위가 없음		 br로 설정된 강제 줄바꿈을 포함
위 아래 글씨 사이 설정, 작성하지 않을 시 기본값 적용

위의 모든 속성은 font 속성 하나로 일렬쓰기가 가능하다

font: font-weight, font-style, font-variant font-size/line-height font-famaily

font-size와 font-family를 필수적으로 작성하여야 일렬 작성을 할 수 있으며
맨 앞 그룹인 꾸밈 속성의 순서는 상관이 없다

글자추가속성

주로 글자나 단어보다는 문장에서 많이 사용된다

  • text-aline
    • 특징 - 글씨를 정렬하는 특징
    • 종류 - center, right, left, justify (양쪽정렬)
    • 주의사항 - block, inline-block 요소에서만 사용가능
    • justify - 양쪽정렬로 내용 안에서 공간이 정해져 있고 두 줄 이상의 글씨에서만 적용된다
  • text-decoration
    • none - 제거
    • underline - 밑줄
    • line-through - 취소선 (가운데선)
    • overline - 윗줄
  • text-transform
    • 특징 - 글자의 형태를 변형하는 유형으로 font-variant 의 확장형 개념으로 보면 된다
    • uppercase - 모두 대문자 변경
    • lowercase - 모두 소문자 변경
    • capitalize - 문장의 첫 글자만 대문자 변경
  • text-indent
    • 특징 - 문단의 첫 머리를 들여쓰기 또는 밀어쓰기 기능
    • 주의사항 - block 또는 inline-block 에만 사용가능
    • 주요사용처 - 대체텍스트를 사용해야할시 마이너스 값을 활용하여 화면에서 벗어나게 사용
  • letter-spacing - 글자와 글자 간격 설정, 글씨 사이에 박스를 넣어준다는 개념 숫자와 단위로 설정, 마이너스 값도 적용 가능
